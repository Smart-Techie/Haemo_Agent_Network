# Haemo_Agent_Network
Haemo-Agent Network is an offline AI coordination system that connects hospitals, donors, and blood inventories instantly using Google's Gemma 4.  When a doctor speaks or types an emergency (e.g., "I need O+ blood"), Gemma extracts the blood type, searches the local network, and returns a complete Emergency Dispatch Report—all without internet.

Key Features:

🎤 Voice Search: Hands-free emergency requests (great for doctors in PPE/bloodied hands).

🩸 Donor Matching: Finds eligible donors based on blood type & 90-day rule.

📦 Inventory Tracking: Real-time local blood fridge stock.

🏥 Regional Network: Connects hospitals within 50km.

⚡ 100% Offline: Runs entirely locally via Ollama + Gemma 4.
